外部スクリプトエディター

 

ラノゲツクールMVでは、一般的なACEコードエディターをベースにした統合型のスクリプトエディターを使用できます。
ただし、他のコードエディターを使用したい場合は、自由にエディターを選んで独自のカスタムスクリプトを作成することができます。

 

プロジェクトの準備

 

外部スクリプトエディターを使用するには、プロジェクトフォルダを開き「scripts」という名前で新しいフォルダを作成します。
このフォルダに全てのカスタムスクリプトを.jsファイルとして格納すると、ラノゲツクールMVはこれらのスクリプトを自動的に認識してゲームに追加します。
スクリプトには以下のように、様々な末尾の数字を追加して組み入れの順番をコントロールすることができます。

 

フルーツ-0.js

りんご-1.js

バナナ-2.js

 

これは、ベースクラスのスクリプトが派生クラスよりも先に組み入れられなければならない場合などで便利です。


プログラミング言語の選択

 

ラノゲツクールMVはスクリプトがJavaScriptプログラミング言語で書かれていることを想定しています。
JavaScriptを使用したくない場合は、CoffeeScript、TypeScript、Rubyなど、JavaScriptにコンパイル可能な別の言語を選択することができます。

 

デフォルトスクリプトの編集

 

外部スクリプトエディターではデフォルトのスクリプトを編集することはできません。これらのスクリプトはラノゲツクールMVのDMS（文書管理システム）で管理しているためです。
しかし、これは通常は問題になりません。アップデートや移行時に変更が失われるため、デフォルトスクリプトを直接編集することは推奨されないからです。

 

デフォルトスクリプトを変更するには、コピーを作成してメソッドや動作を上書きします。これでその変更が移行によって上書きされることはありません。
その後は外部スクリプトエディターを使ってデフォルトスクリプトをスクリプトフォルダにコピーし、変更を加えます。

 

ただし、デフォルトスクリプトを外部ファイルとしてエクスポートする場合は、
「ツール → スクリプトエージェント → Default → ゲームスクリプトエクスポーター」からゲームスクリプトエクスポータースクリプトを使用できます。
このエクスポーターにより、すべてのスクリプトを.jsファイルや.coffeeファイルとしてエクスポートできます。
「JavaScriptとしてエクスポート」チェックボックスにチェックを入れると、コンパイルされたJavaScriptや通常のJavaScriptのみがエクスポートされ、CoffeeScriptコードはエクスポートされません。
