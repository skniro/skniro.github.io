Detailed Description

An utility class which allows you to easily show common message boxes.

To show a simple message box, you can use the following example:

[GSAlert alertWithCaption:@"Information"
message:@"This is a detailed message."
style:CPInformationalAlertStyle
block:function(anAlert, aCode){
console.log("User closed the message box.");
}];

Parameters
aCaptionThe caption-line
aMessageThe detailed message
aStyleThe alert-style
buttonsAn array of texts describing the buttons to display on the message box. The maximum number of buttons is 3:
  • First Button - The highlighted one which reponds to the return-key
  • Second Button - The second button is next to the first button, which is usually the no/cancel button
  • Third Button - The third button is the left-most button.
