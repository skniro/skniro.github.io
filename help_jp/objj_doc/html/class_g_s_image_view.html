Detailed Description

An image view displays a single image and optionally allows the user to select parts of the image with a selection cursor.

This feature is useful to let the user pick a single sprite from a spritesheet or a tile from a tileset. The view can also send an action to the target on double click which is useful to display a dialog where the user can select an image from. Just use CPControl::setTarget: and CPControl::setAction: to setup an action which will be triggered on double-click.
