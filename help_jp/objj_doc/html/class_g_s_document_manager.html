Detailed Description

This class together with GSDocument defines the heart-piece of Visual Novel Maker's Document Management System (DMS).

The document manager stores and manages all documents of the current projects and offers a large set of functions to easily deal with documents.

To access the current project's document manager, use the static defaultManager method. The following example gets all scene documents of the project:

// Get all scene documents
var scenes = [[GSDocumentManager defaultManager] documentsByType:@"vn.scene"];
// Iterate over all scene and print out the name
scene.forEach(function(scene){
// Print out the scene's name. Since the name is part of the document's summary, we
// can access that field even if the scene document has not been loaded yet.
console.log(scene.items.name)
});

- (GSDocument) summary
implementation

The document-index, also called summary.

It is a document which the special UID "SUMMARIES" and contains all summary-items for each document of the project. In most cases you don't need to access this document directly.
