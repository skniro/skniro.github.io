- (id) initWithFrame: (CGRect)  aFrame
collapsible: (BOOL)  collapsible
descriptor: (id)  aDescriptor 
implementation

Initializes a new collapsible view with the specified frame.

Parameters
aFrameThe frame of the new view.
collapsibleIndicates if the header should provide an arrow-button to collapse the view.
aDescriptorA descriptor which contains more info about the CSS classes to use for the header and view. The following properties are supported: (All properties are optional)
  • outline - Indicates if the view should have a border.
  • headerCssClass - The CSS class to use for the view-header.
  • headerTextCssClass - The CSS class to use for the view-header text.
  • headerLineCssClass - The CSS class to use for the view-header bottom-line.
