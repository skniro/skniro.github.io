Detailed Description

This class together with GSResourceManager defines the core of Visual Novel Maker's Resource Management System (RMS).

All project resources, also known as "Assets", like images, music, sounds, etc. are stored and accessed using resources. All resources are stored [project-dir]/resources directory of the project.

+ (GSResourceType) typeFromFolderPath: (CPString)  aPath
implementation

Gets the resource-type for a main folder path.

It checks the first part of the folder-path to determine the resource-type using the following rule-set:

  • If folder-path starts with "Graphics" the resource-type will be "graphic"
  • If folder-path starts with "Audio" the resource-type will be "audio"
  • If folder-path starts with "Movies" or "Videos" the resource-type will be "video"
  • Otherwise the resource-type will be "unknown"
Returns
A resource type.
