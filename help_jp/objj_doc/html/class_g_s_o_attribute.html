Detailed Description

A helper-class to deal with attribute-paths.

An attribute-path describes the path to reach a certain property of an object. The following list shows a few examples:

// Get "System" database record.
var systemData = [[GSODatabase defaultDatabase] systemRecord];
// Accesses the UID of the start-scene
var uid = [GSOAttribute attribute:@"startInfo.scene.uid" ofObject:systemData];
// Accesses the element at index 0 of the colors-array.
var color0 = [GSOAttribute attribute:@"colors.0" ofObject:systemData];
// Accesses the red-value of the color at index 3.
var redValue = [GSOAttribute attribute:@"colors.3.red" ofObject:systemData];

If a property or array-element described in the path does not exist in the object it is automatically created and initialized with null of with an empty-object if sub-objects are defined for it.
