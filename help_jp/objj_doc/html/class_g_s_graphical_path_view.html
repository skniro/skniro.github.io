Detailed Description

An graphical path view allows the user to draw a path using quadratic bezier-curves and then moving objects along that path for example.

A path is constructed from single bezier-curves which are concated to each other. A single bezier-curve has the following structure:

// pt1 - First Point
// pt2 - Second Point
// cpt - Control Point
{ pt1, pt2, cpt }
