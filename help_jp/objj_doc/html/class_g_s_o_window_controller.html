Detailed Description

The default window-controller class to display dialogs/windows defined by JSON view-descriptors such as Custom Views.

It is recommended to GSODialog helper to easily create dialog-boxes from dialog-descriptors defined in JSON.

See Also
GSODialog

To display a Custom View in an own window, you can use the following example:

var windowController = [[GSOWindowController alloc] initWithUid:@"<UID of your Custom View>"];
[windowController setObject:{ value1: 100, value2: "Test" }];
[windowController showModalWindowWithTarget:self didEndSelector:@selector(windowDidEnd:code:)];

There are special UIDs to trigger internal dialog-boxes such as graphic- or variable selection dialog box. An examples look like this:

var windowController = [[GSOWindowController alloc] initWithUid:@"dialog.selectGraphicResource" parameters: { folder: "Graphics/Characters" }];
[windowController showModalWindowWithTarget:self didEndSelector:@selector(windowDidEnd:code:)];

Check out GSOInternalDialogs for more information.

implementation

Initializes a new create window controller with the specified JSON view descriptor.

Parameters
aDescriptorA JSON view descriptor. It is just a JSON/Object, same format as in Extension Editor.
cacheAccessIndicates if the view/window should be restored from cache. In regular, windows/views created from view descriptors are cached to save performance. But if you specify NO, the window/view is recreated and recalculated from the specified descriptor.
