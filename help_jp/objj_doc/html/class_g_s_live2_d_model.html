Detailed Description

A wrapper-class to load a Live2D from a .live2d archive.

If you import a Live2D model intp VN Maker, it will be packed into a .live2d archive for easier handling. This class is just a wrapper around the .live2d archive, it doesn't provide any methods to deal with the Live2D model.
