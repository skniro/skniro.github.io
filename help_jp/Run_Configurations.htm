起動設定

 

既存の起動設定を変更するまたは新しい起動設定を作成するには、メニューバーから「アプリケーションの環境設定」を開いて「実行」タブを選択します。

 

 

このタブにはすべての起動設定が表示されます。各起動設定はダブルクリックすると編集できます。

 

 

  • 「名前」 - メニューや進捗ウィンドウに表示される起動設定の名前です。
  • 「起動フェーズ」 - 連続的に実行される起動フェーズのリストです。

 

各起動設定には、連続的に実行されるフェーズのリストが含まれます。

 

 

  • 「名前」 - 進捗ウィンドウの詳細セクションに表示されるフェーズの名前です。
  • 「コマンドライン」 - ラノゲツクールMVの子プロセスを実行するコマンドラインです。次のフェーズは子プロセスが終了してからトリガーされます。子プロセスの標準的な出力ストリームは、進捗ウィンドウのコンソールにリダイレクトされます。
  • 「作業フォルダー」 - コマンドラインの作業ディレクトリ（CWD）です。空の場合、出力ディレクトリが使用されます。

 

各フェーズはコマンドラインを実行し、指定の方法でゲームデータを処理するかゲームを起動します。コマンドラインでは、「作業ディレクトリ」フィールドと同様に、「$(NODE)」のように「$」から始まる変数のセットを使用できます。使用可能な変数とその値の詳細については、下記を参照してください。

 

変数

 

起動設定フェーズのコマンドラインや「作業ディレクトリ」フィールドでは、次の変数を使用できます。変数は次の形式である必要があります。

 

$(VARIABLE_NAME)

 

使用できる変数は次のとおりです。

 

  • NODE - ラノゲツクールMV内部のNode.js実行ファイルへのパスです。
  • APP_DIR - ラノゲツクールMVのインストールディレクトリへのパスです。
  • PROJECT_DIR - 現在のプロジェクトフォルダーへのパスです。
  • OUTPUT_DIR - 出力フォルダーへのパスです。
  • PROJECT_NAME - 現在のプロジェクトの名前です。
  • GAME_TITLE - プロジェクトのゲームタイトルです。
