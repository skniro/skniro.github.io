ゲーム内ユーザーインターフェイス

 

ご注意: 本項はゲーム内ユーザーインターフェイスの作り方や既存のインターフェイスの変更方法をご説明するものではありません。それらについてのチュートリアルは IGUI Getting Started ガイドや Beginner's Guide をお読みください。

 

本項では、ラノゲツクールMV のゲーム内ユーザーインターフェイスがどのように構築されているか技術的な側面から確認し、より深い部分での変更に必要な情報を提供します。

 

UI オブジェクト

 

ラノゲツクールMV のゲーム内ユーザーインターフェイスは、 テキストラベルを表示する ui.Object_Text、イメージを表示する ui.Object_Image、フレームを表示する ui.Object_Frame などといった ui.xxxx のゲームオブジェクトの集合体です。また、 ui.Object_StackLayout や ui.Object_GridLayout のようなレイアウトオブジェクトも含まれます。それらのレイアウトオブジェクトは、ピクセル座標を使わずに画面上に UI 要素を簡単に配置するために使用されます。

 

UI ゲームオブジェクトは非常に基本的なオブジェクトで、 ui.Object_Button オブジェクトとかそういうものはありません。ラノゲツクールMV の UI システムは、基本的な UI 要素を組み合わせて高度な要素を作成するように設計されています。例えば、ボタンなら、 ui.Object_Image と ui.Object_Text と組み合わせて作ります。もちろん自分でカスタム UI オブジェクトやカスタム UI コンポーネントを作成することも可能です。既定の UI オブジェクトは、 Objects > UI フォルダーに、UI コンポーネントは Components > UI フォルダーにそれぞれ格納されています。

 

JSON レイアウト、テンプレートおよびスタイル

 

ラノゲツクールMV では、既定のゲーム内ユーザーインターフェイスは、コード内で UI オブジェクトを直接処理するのではなく、 JSON レイアウト、スタイルおよびテンプレートを使用して作成されています。JSON ベースのレイアウトと追加システムはベーシックなゲーム内ユーザーインターフェイスシステムの上に構築されています。この背景には、使いやすい JASON descriptions を使ってゲーム内ユーザーインターフェイスをより簡単に作成する狙いがあります。JSON structure は ui.UIManager クラスによってパース、処理され、 JSON structure の情報をベースに必要な UI オブジェクトをすべて構築します。

 

それにより、プログラミングと格闘する必要はなくなり、初心者でもウェブサイトをデザインするような感覚でゲーム内ユーザーインターフェイスを簡単に変更できるようになります。ほとんどの場合、プログラミングなしですべてのゲーム内ユーザーインターフェイスを変更可能です。プログラミングが必要になるのは、機能を追加したり、動きを大幅に変更する場合のみになるでしょう。

 

すべてのレイアウト、テンプレートおよびスタイルはスクリプトエディタの 「Layouts」 フォルダー内に格納されます。デザインの変更は、 Layout_XXX スクリプトや Template_XXX スクリプトの編集で簡単に実現できます。フォントや色、アニメーションの変更は、 Style_Default スクリプトを編集します。

 

UIManager クラス

 

UIManager クラスは JSON レイアウト、テンプレートおよびスタイルのパースと処理を助けます。必要な UI オブジェクトを構築し、それらに必要なコンポーネントを追加して適切なスタイルを適用します。これはすでに「例: 移動可能なキャラクター」の項で既に確認しています。その例では最初に ui.Object_Text を直接使用し、その後 JSON レイアウトに移ってキャラクター情報表示を作成しています。また UIManager.createFromDescript メソッドを使用してパースされた JSON description からレイアウトを作成しています。

 

UIManager には様々な種類のメソッドがあります。自由にお試しいただき、オリジナルの UI オブジェクトを作成してみてください。ご不明な点はフォーラムでお尋ねください。ゲーム内ユーザーインターフェイスのドキュメント もお読みください。
