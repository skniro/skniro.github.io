Properties

items.category

Strings

Database category the data record belongs to.

items.data

Object

Contains actual data of the record.

items.data.booleanVariables

gs.Variable

Stores all local boolean variable definitions by index.

items.data.index

Object

Index of the data-record in database used in-game. Starts with 0. So 0 = Record 0001, 1 = Record 0002, etc.

items.data.listVariables

gs.Variable

Stores all local list variable definitions by index.

items.data.name

Object

Name of the data-record used in-game.

items.data.numberVariables

gs.Variable

Stores all local number variable definitions by index.

items.data.stringVariables

gs.Variable

Stores all local string variable definitions by index.

items.id

Strings

ID of the data record which matches with the data.index.

items.isFolder

Boolean

Indiciates if the data record is a folder.

items.localizableStrings

Object

Contains all texts used in the scene which can be translated through language configuration view/editor. See LocalizableString documentation.

items.parentId

String

UID of the data-record's parent-document. That can be data-record configured as folder or null if its a top-level data record.
