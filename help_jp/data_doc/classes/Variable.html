Properties

index

Number

The variable's index. Means which variable is this. So index 0 = Variable 0001, 1 = Variable 0002, etc.

name

String

The variable's display name.

scope

Number

The variable's scope.

  • 0 = Local Variable
  • 1 = Global Variable
  • 2 = Persistent Variable
