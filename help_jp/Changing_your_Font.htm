ゲーム内フォントの変更

 

ゲーム内で使うフォントを変更することで、ゲームが一層個性的な作品になります。一貫性を保つため、フォントを変更した方がいいであろう4つのエリアに気を配ることが重要です。

 

  • アドベンチャー（ADV）メッセージテキスト - 既定のメッセージボックスを使用する場合
  • ノベル（NVL）メッセージテキスト - NVLコマンドを使用する場合
  • バックログのメッセージテキスト - バックログをチェックする場合
  • カスタムメッセージテキスト - 「メッセージエリアを設定」を使用する場合

 

フォントの変更方法は次のようにいくつもあります。

  • テキストコード
  • メッセージ設定
  • スクリプトの呼び出し
  • スクリプトエディタ

 

スクリプトエディタの使用

勇気がある方は、スクリプトエディタを使ってフォントを変更してみましょう。

 

  • 「スクリプトエディタ」を開きます。
  • 右クリックから->「新規フォルダー」を選択し、次のように「Main」の上に新しいフォルダーを作ります。
  • 右クリックから->「新規フォルダー」->「プロパティ」の順に選択し、任意の名前を付けます。
  • 作成したばかりのフォルダーを選択します。右クリックから->「新規 CoffeeScript ファイル」の順に選びます。
  • 右クリックから->「新規 CoffeeScript ファイル」->「プロパティ」の順に選択します。「Style_New」と名付けましょう。
  • そこに次のコードを挿入してください。
###
# CONSTANTS FOR FONT SIZE
###
class DesktopUIConstants
    @TEXT_SIZE_SMALL = 20
    @TEXT_SIZE_MESSAGE = 24
    @TEXT_SIZE_MESSAGE_NAME = 25
gs.DesktopUIConstants = DesktopUIConstants
gs.UIConstants = DesktopUIConstants
###
# NAME BOX
###
ui.UIManager.styles.messageBoxNameText = {
    "font": { 
        "name": "Times New Roman", 
        "size": gs.UIConstants.TEXT_SIZE_MESSAGE_NAME, 
        "smallCaps": false, 
        "italic": false, 
        "color": [59, 59, 59, 255],
    }
}
###
# ADV MODE
###
ui.UIManager.styles.advMessageText = {
    "font": { 
        "name": "Times New Roman", 
        "size": gs.UIConstants.TEXT_SIZE_MESSAGE, 
        "smallCaps": false, 
        "italic": false, 
        "color": [59, 59, 59, 255],
    }
}
###
# NVL MODE
###
ui.UIManager.styles.nvlMessageText = {
    "font": { 
        "name": "Times New Roman", 
        "size": gs.UIConstants.TEXT_SIZE_MESSAGE, 
        "smallCaps": false, 
        "italic": false, 
        "color": [59, 59, 59, 255],
        
    }
}
###
# CUSTOM MESSAGE
###
ui.UIManager.styles.customMessageText = {
    "font": { 
        "name": "Times New Roman", 
        "size": gs.UIConstants.TEXT_SIZE_MESSAGE, 
        "smallCaps": false, 
        "italic": false, 
        "color": [59, 59, 59, 255],
        } 
}
###
# BACKLOG, ETC.
###
ui.UIManager.styles.messageText = {
    "font": { 
    "name": "Times New Roman", 
    "size": gs.UIConstants.TEXT_SIZE_MESSAGE, 
    "smallCaps": false, 
    "italic": false, 
    }
}
###
# Defines the font used for the name-column of the message backlog.
###
ui.UIManager.styles.backlogNameText = {
    "font": { 
        "name": "Times New Roman", 
        "size": gs.UIConstants.TEXT_SIZE_MESSAGE_NAME, 
        "smallCaps": false, 
        "italic": false, 
        "border": true, 
        "borderSize": 4 }
}

簡単に説明します。

  • name - フォントフォルダーにインポートしたフォントの名前。Times New RomanやArialなど、システムに既定でインストールされているフォントを使うこともできます。
  • size - フォントサイズ。一定ではなくサイズを指定したい場合、この値を数値に変更することができます。
  • TEXT_SIZE_MESSAGE - メッセージフォントの既定のサイズ
  • TEXT_SIZE_MESSAGE_NAME - 名前フォントの既定のサイズ
  • smallCaps - すべてのテキストを小文字に変更
  • italic - フォントをイタリック化
  • outline - フォントのアウトラインの色とサイズ

 

独自のプロパティを追加することもできます。例えば、メッセージに太字を追加すると、次のようになります。

###
# ADV MODE
###
ui.UIManager.styles.advMessageText = {
    "font": { 
        "name": "Times New Roman", 
        "size": gs.UIConstants.TEXT_SIZE_MESSAGE, 
        "smallCaps": false, 
        "italic": false, 
        "bold": true,


        "color": [59, 59, 59, 255],
    }
}
