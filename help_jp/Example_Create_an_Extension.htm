例：エクステンションの作成

 

さて、今度はいよいよエクステンションコンポーザーを使用してカードのエクステンションを作成し、他の人々と共有することができるようになります。

エクステンションを作成するには、「ツール」 → 「デベロッパーツール」 → 「エクステンションを作成」のメニューからエクステンションコンポーザーを開いてください。

  • ドキュメント

エクステンションによって変更可能なすべてのドキュメントを表示します。エクステンションエディタのすべてのドキュメントや、ゲームスクリプト、シーン、データベースがこれに相当します。

ここで作成または変更したドキュメントを選択し、「追加」ボタンをクリックしてエクステンションに加えることができます。

 

  • あなたのエクステンション

エクステンションの内容を表示します。「ドキュメント」ビュー内のドキュメントを選択し、「追加」ボタンをクリックしてエクステンションへと簡単に加えることができます。

エクステンションからドキュメントを削除するには、削除するドキュメントまたはフォルダーを選択して「削除」ボタンをクリックしてください。

 

  • ID

エクステンションの固有の識別子。所属する企業またはウェブサイトのドメインのような、固有のものを使います。（例：com.example.ExampleExtension）

今回は「com.degica.ExampleCardExtension」を使います。

 

  • 組織

組織や会社の名前

 

  • 名前

エクステンションの名前。ここでは「Card Extension」と名付けることにします。

 

  • バージョン

エクステンションのバージョン。今回は「1.0.0.0」を使います。

 

  • 作者

エクステンションの作者または開発者。今回は「DEGICA」を使います。

 

  • ウェブサイト

作者または開発者のウェブサイト。エクステンションに関する情報をユーザーが参照できるよう、あなたのウェブサイトへのリンクをここに入力してください。

今回は「www.degica.com」を挿入します。

 

  • スクリプトをインストール

エクステンションにオプションのインストールスクリプトを使用することができます。以前のバージョンのエクステンションで作成されたデータレコードまたはシーンコマンドを移行する必要がある場合に便利です。
詳しくは、エクステンションシステムのリファレンスを参照してください。今回は空白にしておきます。

 

  • スクリプトをアンインストール

クリーンアップの必要がある場合、オプションのアンインストールスクリプトをエクステンションに使用することができます。

詳しくは、エクステンションシステムのリファレンスを参照してください。今回は空白にしておきます。

 

  • 説明

エクステンションが何をするか、エディターやゲームエンジンにどんな機能を追加するかなど、詳細な説明を含めます。今回は次のテキストを追加します。

「カードを使ったちょっとしたミニゲームに使える、シンプルなカードをユーザーがデータベースに作成できるエクステンション。

カードをスクリーンに表示するためのシンプルなシーンコマンド「Show Card」（カードを表示）も提供。」

 

  • ライセンス

ライセンスの使用許諾契約書をここに入力することができます。エクステンションをインストールする前に、ユーザーはこれに同意する必要があります。今回は空白にしておきます。

 

では、作成したドキュメントをエクステンションに追加しましょう。作成したドキュメントは、データレコードビュー「Cards」、新しいシーンコマンド「Show Card」（カードを表示）、

そしてインタプリンタを拡張するゲームスクリプト「Component_CommandInterpreterCardExtension」の3点です。これらを選択して「追加」ボタンをクリックするとエクステンションに追加されます。

これで、エクステンションコンテンツは次のようになっているはずです。

これができたら「作成」ボタンをクリックし、エクステンションが作成される場所を選択したら完了です！

 

エクステンションコンポーザーは、選択した場所にエクステンション名が付いた.zipアーカイブを作成します。このzipアーカイブを共有すれば、

他のユーザーも簡単にエクステンションマネージャーを使ってこのエクステンションをインストールすることができます！
