一括編集

 

「一括編集」はチャプターやシーン内の特定のコマンドを一括で変更できるメニューです。プログラミングの経験があるユーザーであれば、さらに多くの操作ができます。.

 

「コマンド」

任意のシーンコマンドを選択して変更できます。

 

「数値フィールド」

コマンドの数値フィールド（「待機」など）を変更します。

 

  • 「設定」 - 現在の値を上書きします。

  • 「増加」 - 現在の値を増やします。

  • 「減少」 - 現在の値を減らします。

 

「テキストフィールド」

コマンドのテキストフィールド（「メッセージを表示」など）を変更します。デフォルトの言語設定のみが対象です。翻訳を編集するには「言語設定」タブを使用する必要があります。

 

  • 「設定」 - 現在の内容を設定します。

  • 「追加」 - コマンドの現在の内容の最後に追加します。

  • 「削除」 - 入力されたキーワードを検索して削除します。

 

「スクリプト」

プログラミングを使用し、「正規表現」など、詳細な条件や置換を指定のコマンドに作成できます。内部では、スクリプトコードを含んだJavaScript関数が作成されます。

 

function(command) { Your Code is inserted here }

 

関数は現在のコマンドをパラメータとして取得するため、ローカルの「command」変数を使用してスクリプト内のコマンドを確認および変更できます。詳細やコマンドのデータ構造については、ラノゲツクールMVのドキュメント「データ形式」を参照してください。

 

「条件」

一括編集の条件を設定します。
