「ラノゲツクールMV」について

ラノゲツクールMV（英語名：Visual Novel Maker）はビジュアルノベルおよびそのサブジャンルの制作に特化したエンジンです。コンピューターおよびモバイルデバイス向けに、対話型の物語を創作するユーザーをサポートします。

使いやすいエディターで、プログラミングの知識がなくても簡単にビジュアルノベルを制作することができます。ラノゲツクールMVはAndré Radomskiによって開発されました。

 

主な特長

ラノゲツクールMVには強力かつ使いやすいインターフェイスとともに、柔軟かつ高度な機能が備わっています。

 

マルチプラットフォームをサポート

Windows、MacOSX、Linux Ubuntuでゲームの作成が可能です。

 

簡単に理解できるシーンエディタ

シーンエディタでオリジナルのゲームのシーンを簡単に作成できます。

 

合理化されたデータベース

データベースにより、ユーザーは自由にデータを整理できます。また、フォルダー内のデータを簡単に並び替えることができます。

 

高度なスクリプト記述およびプラグインシステム

CoffeeScriptまたはJavaScript言語によるスクリプト記述がサポートされています。ゲームの見た目をカスタマイズしたり、ゲームシステム全体を変更したりすることが簡単に行えます。

さらに機能が必要になったら、エディターをカスタマイズし、パワフルなプラグインシステムで機能を追加することもできます。

 

言語設定

自分のゲームを他の言語に翻訳したい場合、この機能を使えば簡単にできます。

 

マルチゲームパッケージオプション

複数のOSおよびデバイス向けにゲームを構築することができます。

  • HTML5/ウェブ
  • Windows 32/64ビット
  • Linux Ubuntu 32/64ビット
  • OS X 32/64ビット
  • SteamOS
  • Android
  • iOS

 

最低システム要件

ラノゲツクールMVの最低システム要件は次の通りです。

 

OS

 Windows 7/8.1/10（64ビット）、Mac OS X 10.10（64ビット）、Linux Ubuntu（64ビット）

プロセッサ

 Intel Core i7以上

グラフィック

 DirectX/OpenGLをサポート可能なGPU

ディスプレイ

 1280x768以上のディスプレイ

ストレージ

 3GB 利用可能

特記事項

 OpenGLの高速レンダリングにはOpenGLのハードウェアドライバサポートが必要です。

 

  • ゲームのパフォーマンスは、作成するゲームとユーザーの環境に大きく依存します。
