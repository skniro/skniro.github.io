スクリプターズ ガイド

 

ラノゲツクールMV には、強力なスクリプト作成機能が用意されており、必要に合わせてゲームエンジンに変更を加えることが可能です。本ヘルプファイルにはスクリプトとプラグインの詳細な資料を収録し、そこでは API の説明や用例を確認できます。

 

本スクリプターズ ガイドは、ラノゲツクールMVのゲームスクリプト作成システムについて、最初の一歩を踏み出すお手伝いをします。何か質問したいこと、お困りのことがございましたら、フォーラムに投稿をお願いいたします。

 

ラノゲツクールMVのスクリプト作成

 

ビジュアルノベルを制作しようとしているあなたが、「スクリプト作成」という言葉から想像するのは以下のようなものではないでしょうか？


show image "background.jpg" 
show image "character_nina.png" at center 
nina = create character "Nina", Color.Blue
nina "Hey! How is it going!" 
if $already_seen
  nina "Ah... its just you..."
else
  nina "So nice to meet you!"

これは架空の例なのですが、従来のビジュアルノベルのスクリプトエンジンはグラフィカルユーザーインターフェイスを持たず、そこでのスクリプト作成は確かにこのようなものでした。しかし、 ラノゲツクールMVのスクリプト作成システムは全く異なります。ラノゲツクールMVでのスクリプト作成とは、実際のビジュアルノベルのゲームエンジンを変更したり拡張したりすることを意味するもので、ビジュアルノベルのシーンを一行一行書いていくものではありません。

 

ラノゲツクールMVでのスクリプト作成とは、そういった類の作業のベースになるものです。つまり、 ラノゲツクールMVのゲームエンジンについて学べば、従来型のスクリプトエンジンを自分で作れてしまうかもしれない、ということです。ただし、 ラノゲツクールMVには、 シーンエディタ という強力で拡張性に優れたグラフィカルエディタが用意されていますので、 シーンのスクリプト作成にはこちらを利用して ラノゲツクールMVの全機能を最大限活用してください。

 

本ガイドについて

 

本ガイドでは、 ラノゲツクールMVのベーシックゲームエンジンとオブジェクトコンポーネントシステムおよび実際の VN エンジンの基本についてご説明しますが、プラグインの作り方やシーンエディタコマンドのカスタマイズ、ゲーム内ユーザーインターフェイスの変更方法などを具体的にお教えするものではありません。これらについてお知りになりたい場合は、次の項目をご確認ください:

 

エクステンションのドキュメント + 用例 (エディタやゲームエンジン用プラグインの作り方)

ゲーム内ユーザーインターフェイスシステムのドキュメント + 用例 (ゲーム内ユーザーインターフェイスのレイアウト変更方法など)
