はじめに

 

プロジェクトの作成

最初にプロジェクトを作成する必要があります。プロジェクトとは、ゲームを構成するデータとリソースを集めたものです。リソースマネージャーを使用すると、ファイルをインポートしてプロジェクトに画像や音楽を追加することもできます。プロジェクトの作成方法は以下のとおりです。

 

（1）「ファイル」 -> 「新規プロジェクト」クリックします。

 

 

（2）「新規プロジェクト」ウィンドウが表示されます。

 

 

（A）「プロジェクト名」 - プロジェクトのフォルダーの名前

（B）「ゲームタイトル」 - ゲームウィンドウに表示されるプロジェクトのタイトル

（C）「テンプレート」 - さまざまなオプションでゲームを簡単に設定できる既定のプロジェクト

（D）「場所」 - プロジェクトを保存するフォルダーのパス

 

（3）「処理中」ウィンドウが閉じるまで待機します。

 

 

（4）プロジェクトが作成されました。これでゲームの作成を開始できます。

 

 

プロジェクトの管理

保存/ロード

プロジェクトの作業を中断するまたはプログラムを終了する場合は、事前に必ずプロジェクトを保存してください。「保存」ボタン（または「ファイル」 → 「プロジェクトを保存」をクリックすると、現在作業中のプロジェクトが保存されます。この操作により、同じプロジェクトの既存のデータはすべて上書きされます。

 

プロジェクトを開く

プロジェクトは次の2つの方法で開くことができます。

  • ホームビュー画面の「プロジェクトリスト」でプロジェクトを選択してダブルクリックします。

  • 「プロジェクトを開く」ボタン（または「ファイル」 → 「プロジェクトを開く」をクリックし、目的のプロジェクトのフォルダーを選択します。

 

移行システム

プログラムの更新後に新しいプロジェクトを作成するまたは新しいプロジェクトを開くと、「移行」ウィンドウが表示される場合があります。

 

移行とはプロジェクトを最新バージョンに更新することです。移行により、古いバージョンのバグなどの問題が修正されます。移行せずにプロジェクトを開くことはできません。

 

プロジェクトのバックアップ／削除

プロジェクトのコンテンツは、プロジェクト作成時に指定されたフォルダーにまとめて保存されます。バックアップを取るには、フォルダー全体を別のハードディスクやリムーバブルメディアなどにコピーします。不要になったプロジェクトを削除するには、「プロジェクトリスト」にあるものであれば「プロジェクトを削除」をクリックします。リストにないプロジェクトの場合は、通常の操作でプロジェクトフォルダーを削除します。
