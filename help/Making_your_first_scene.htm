Make your First Scene

 

In this tutorial we'll show you how to create a basic scene. Make sure to check Adding Chapters and Scenes page so you know how to create Chapters and Scenes! For this tutorial, we're going to use the default sample project.

 

  • Inside the Test Bed Chapter, let's create a New Scene called My First Scene.

 

Adding Commands

When you first open Visual Novel Maker, you will notice the Commands panel over here:

 

In order to navigate the commands, just press the little icons and you will see the list of commands. There are many ways to add the commands. For example, if we want to add a Switch in a scene, click Basic co mmand -- if you are not in that section by default. You can add the Switch command by clicking, drag and drop or typing Switch on Scene Content.

 

Now that we know how to add commands, let's make our first scene! Remember, you can test your scene through Live Preview as you are adding commands.

 

 

Adding a Background

  • Click the Background command.
  • Add Change Background. Click File and select a background.
  • You will notice in the Live Preview that the animation is going from the left. Click the arrow beside Change Background and we will see extended options. For a complete explanation of these commands, just go to the Change Background link and read over there. But for now, we'll be changing the Animation.
  • Click Animation and select Blending.

 

Adding Background Sound

  • Click the Audio Playback command.
  • Add Play Music command. Click File and select a track. For more information about each command, simply refer to the page.

 

Adding a Character in the Scene

  • Click the Character Command.
  • Select Join Scene and select a character. By default we're using Ace and change the animation to blending mode.
  • To move the character in the center of the screen, either use Direct or Predefined -> Top. Here's the setting in case you get lost.
  • Now that character will appear on our Scene!

 

Adding Dialogue

  • Click the Message Command.
  • Select Message Box Visibility and change Visible to Yes.
  • Add the Show Message Command. Change the Character to Ace.
  • Click on Insert message text here in order to enter text! It can be anything!

 

 

If you did the steps above correctly, the scene should look similar to this:

 

Testing your Scene!

Now that you are done with your First Scene, it's time to Playtest!

  • Right-click the Scene and select Set as Start Scene.  
  • Save and press the Play button!
  • A small prompt will appear, this only ensures that you saved before testing. Just click Yes.
  • Now you can test your game!

 

Note:

  • If you want to skip the Language Screen and Title Screen, Right-click the Scene and select Set as Intro Scene.

  • You can change the Intro and Start Scene on Database -> System Tab.

 

Congratulations you just created your first Scene!
