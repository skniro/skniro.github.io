GSQTextArea Class

A text input control which allow the user to input a text value.

  • Index
  • Properties

Item Index

Properties

  • localizable
  • multiline

Properties

localizable

Boolean

Indicates that the content of the text-field can be translated into different languages by the user using the language configuration view.

Default: false

multiline

Boolean

Indiciates if the text-field is multiline. Only works correctly if the text-field placed in fullSizeItems. A multiline text-area cannot be calculated by variable using variableButton property.

Default: false
