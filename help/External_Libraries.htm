External Libraries

 

Visual Novel Maker allows you to add any external third-party JavaScript libraries if you need them for your game.

 

If you want to add an external library, just navigate to your project folder and create a new folder "lib" there and put all your external libraries there. All external libraries are automatically included before the actual game scripts.

 

Important Notes

 

Keep in mind that if you add any external libraries, your game might lose portability because that libraries might not be available on all platforms. Especially if your library operates with HTML5 APIs because if those APIs are not available on a certain target platform, you game might not run there.
