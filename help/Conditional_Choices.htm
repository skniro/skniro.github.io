Conditional Choices

 

In this tutorial, we'll teach you how to make dynamic choices. This means that certain choices won't appear until you trigger a flag. Since this is an advanced concept, we do not recommend trying this tutorial until you understand how Variables, Switches, Labels and Conditions work.

 

Here's the situation, the character is trapped in a room and has to investigate objects. In this case just the bed, door and windows. The only thing is, we want the Windows to appear after we check the door. So here's how the process will work:

So here's a quick breakdown:

  • In order to make choices appear based on certain actions, we have to put the choices and prompts under Conditions!
  • On the Add Choice: Window it says it will show itself to the player if the Local Switch #1 is Equal to ON.
  • Since in the flow chart we see that the window option is unlocked when you check the door, we added another condition there that checks if the switch is OFF. After that, it will give you a prompt and turn on a switch. Should the player check the door again, it will not return this prompt making it a one-time thing only.
