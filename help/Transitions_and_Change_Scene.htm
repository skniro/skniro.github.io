Transitions and Change Scene

 

Now that we covered the basics and some essential knowledge, it's time to learn how to move from one scene to the next.

 

Basic Change Scene

A basic change scene by default is a basic Fade-Out.

  • To add the command, go to Scene -> Change Scene.
  • Click on the 'Scene' tab of said command and just select the Scene you want to go to.

 

Fancy Change Scene

A fancy change scene involves the use of transition masks. This changes your scene dynamically to another. For example, if you want a scene looks like it's 'blinking' then you would use a transition mask that looks like an eyelid.

 

  • First, add the command Scene -> Change Transition.  For more information about each individual command, please check the page link.
  • Add the command Scene -> Change Scene right after Change Transition.

 

Remember!

  • Change Scene and Call Scene are two completely different commands.
  • Change Transition is global! If you want the change scene use the classic fade in and fade out, just use another change transition and set the file to none.
  • If you find yourself with a scene containing a BGM and wanting to remove it, you will need to manually add the command  Audio Playback -> Stop Music before the Change Scene command.
