SceneCommand Class

Describes a single scene command.

Properties

expanded

Boolean

Indicates if the command is expanded in scene editor or not.

id

String

Identifier for the command so the command interpreter can recognize how to interpret this command.

indent

Number

Indentation level

params

Object

Stores the command-specific data. See Scene Commands in Extension-Editor for more information about which command saves what kind of specific data.

uid

String

Unique Identifier (UID) of the command. That UID is necessary for certain multi-language features.
