The Context Menu

 

The Context Menu is also known as drop down or right-click menu. It can be accessed through Right-Click () in Windows and (+Click) in Mac OS X.

 

General Commands

Cut

Copies and removes the selected element(s) from its current position.

 

Copy

Copies selected element(s) or keyword(s).

 

Paste

Adds the content of the clipboard below the selected command.

 

Delete

Deletes the currently selected element(s).

 

Scenes Commands

New Chapter

This command only appear if a Chapter Title is highlighted. This creates a new chapter.

 

Delete Chapter

This command only appear if a Chapter Title is highlighted. Deletes the currently selected chapter.

 

New Scene

Creates a new Scene. You cannot create a new scene if there's no chapter.

 

Set as Start Scene

This command only appear if a Scene is highlighted. This command sets a scene as Start Scene seen in System collection.

 

Set as Intro Scene

This command only appear if a Scene is highlighted. This command sets a scene as Intro Scene seen in System collection.

 

Copy

Copies selected command(s) in JSON format.

 

Copy as Text

Copies selected command(s) in plain text. Useful for sharing in forums, chats, etc.

 

Properties...

This command only appear if a Scene is highlighted. This allows the user to edit a scene's properties.

  • Name - The current name of the Scene. Feel free to rename it.

  • Disable Live-Preview - Disables Live-Preview for this Scene. This is useful for special cases where a Live-Preview of the Scene's content makes no sense.

 

Mass Edit...

Allows you to edit all commands in a scene. An in-depth explanation can be found in Mass Edit.

If a Chapter is selected, it will edit the command you set in all scenes and sub-scenes under that chapter.

If a Scene is selected, it will edit the command you set only on that scene.

 

Collections

New Record

Creates a New Record.

 

New Folder

Creates a New Folder.

 

Copy

Copies the selected Record or Folder.

 

Paste

Adds the copied Record to the list.

 

Paste as Overwrite

Replaces the selected Record instead of creating a new Record.

 

Delete

Deletes the currently selected Record or Folder.

 

Clear

Empties the content of the selected Record.

 

Sort by ID

Sorts Records inside a Folder by ID.

 

Language Configuration

New Language

Creates a new Language Profile.

 

Edit Language

Opens the Language Profile window.

 

 

Refresh Texts

Collects all localizable texts from the selected scene or database record to allow translation. If you trigger this action on the Language Profile element, all localizable texts from all scenes and the entire database will be collected. Whenever you change, add or remove texts to your game you have to trigger this command. Otherwise you will not see them in Language Configuration.

 

Import CSV

Imports texts from CSV files into the selected language profile. The import process is recursive, it searches in all sub-folders for possible CSV files to import.

 

Export CSV

Exports all texts as CSV files from the selected language profile.
