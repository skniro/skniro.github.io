Animations

 

 

General

  • Name - The name of the animation.

 

Settings

  • Graphic - The file of the animation.

  • Frame Grid - The number of frames in a row (X) and column (Y).

  • Frame Range - The amount of frames the animation has starting from 0 to it's maximum. If you have 30 frames, then it's 0~29.

  • Repetition Range - Starting and ending frames for looping the animation.

  • Duration - The length of the animation.

  • Repeats - If checked, the animation will loop.
