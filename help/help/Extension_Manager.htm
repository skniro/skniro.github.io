Extension Manager

 

Visual Novel Maker provides a powerful extension system which allows the user to add new features to the editor and game engine. Extensions can be easily installed/uninstalled using the Extension Manager. Since extensions can make heavy changes to the editor or game engine, it is highly recommended to make a backup of your project before installing an extension.

 

Install an Extension

You can open the Extension Manager through Menu under Tools > Extension Manager.

To install an extension, just click the"Install " button and select the extension-file(*.zip) you like to install. Confirm that you really want to install the extension and read & accept the license agreement of the extension (if any). After the extension has been successfully installed it will appear in the"Extensions " list.

 

If you select an extension, you can see the following information.

 

  • ID - The unique identifier of the extension. That is for technical reason to unique identify each extension & developer independent from the extension’s name.

  • Organization - The name of your company or organization.

  • Name - The name of the extension.

  • Version - The version of the extension. Check the developers website if there is maybe a newer version available.

  • Author - The author/developer of the extension.

  • Website - The author’s/developer’s website. In regular you can find more information about the extension there.

  • Description - Contains a more detailed description about what the extension is doing, what kind of features it adds to the editor and/or game engine, etc.

 

You can click the"View License " button to review the license/terms & conditions of the extension.

 

Activate/Deactivate an Extension

An installed extension can be deactivated anytime by unchecking the "Active " checkbox. If you deactivate an extension, all changes to the editor and game engine are removed and the extension doesn’t influence the editor/game anymore. But the resources are not deleted and the extension is still installed and can be re-activated anytime. It is recommended to restart Visual Novel Maker after deactivating/activating an extension.

 

Uninstall an Extension

To uninstall an extension, just click the"Uninstall " button and confirm that you really want to uninstall the extension. Uninstalling an extension removes the whole extension with all its installed data  and even all extension-specific content created by the user. So if you uninstall an extension which added a new database category, all your created records will be deleted as well. It is recommended to restart the Visual Novel Maker after uninstalling an extension.
