Mini Games

 

With Visual Novel Maker, you have the power to create own mini games for your Visual Novel easily once you mastered the basics of it. Before you start making mini games, make sure that you read and understand the entire Beginner's Guide. Also, feel free to check out our Forums for help!

 

Make a Mini Game without Scripting

 

In VN Maker, you can create mini games without programming. You can just create a scene and use pictures, hotspots, music and sounds to make your mini game. For the mini game logic, it is recommended to use Common Events and/or sub-scenes (see Call Scene command) to keep the overview.

 

One example is the Black Jack Example game you can study to learn more about how to make mini games without scripting.

 

 

 

 

Make a Mini Game using Scripting

 

If you have experience with programming, you can also create mini games with real programming if necessary. In that case, you should read the entire Scripter's Guide first and also check out the Engine API Documentation as well as the GameScript API Documentation.
