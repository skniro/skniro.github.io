Custom Resource Processors


Visual Novel Maker supports the most common graphic and audio formats by default. However, if you are working on a more complex extension, you maybe need to add new resource/file types to Visual Novel Maker. You can do that easily be defining a custom resource pipeline processor using Objective-J.

 

You can find an example resource processor in Extension Editor > Scripts > Resource Processors > GSFntResourcePipelineProcessor which adds support for .fnt files. For more information, check out the Objective-J API Reference. Don't hesitate to ask for help in our forums.

 

To create a custom main folder in Resource Manager, you just need to edit <project-dir>/resources/folders.json during the install-script of your extension.
