Live2D Commands

 

The Live2D commands are used to display Live2D models on the screen.

 

Important Notes

  • The commands [Move Character], [Move Character along Path], [Zoom Character], [Shake Character] and [Blend Character] from Characters category affect Live2D models too.

 

Live2D Join Scene

Lets a Live2D Character join the scene.

  • Character - The character to join.

  • Model - The Live2D model to use.

  • Position - The position of the Live2D character.

  • Predefined allows to select one of the predefined positions configured in Database > System > Predefined Object Positions.

  • Direct allows you to set the position using the graphical editor.

  • Calculated allows you to calculate the position by variable.

  • Duration - The duration of the join-animation in milliseconds.

  • Continue will make the scene immediately continue.

  • Wait will make the scene wait until the join animation is done.

  • Effects - These options affect how the Character will appear.

  • Easing sets the tween movement of the character. More information can be seen in Easing Effects page.

  • Motion Blur - Set Yes to apply motion blur, No to turn it off.

    • Spawn Time - The amount of frames after a new copy is created.. By default, the amount is 2.

    • Dissolve Speed - The rate at which the spawned copies dissipate. Specifically, the opacity of the copies are dropped every frame.

    • Opacity - The opacity the copied image starts at the time it is spawned. By default, it is at 100.
      For example, if the dissolve speed is 2, the shadow/ghost/particle is completely invisible/dead after 50 frames.

  • Display - Allows you to specify the display order and anchor for the character.

  • Anchor sets the anchor point of your character between top-left and center. Setting the anchor to center is only important when the zoom and rotation commands will used and will not affect positioning.

  • Z-Order sets the display order of the characters.

  • Viewport - The viewport the character will be assigned to. The viewport determines if the character gets affected by camera-movements or screen effects or not.

    • Scene - The character is assigned to the scene-viewport and behaves like an object which belongs to the scene so it gets affected by camera-movements and screen effects.

    • User Interface - The character is assigned to the ui-viewport and behaves like an object which belongs to the user interface so it doesn't get affected by camera-movements and screen effects.

 

Live2D Exit Scene

Lets a Live2D Character exit the scene.

  • Character - The character to remove from scene.

  • Duration - The duration of the exit animation in milliseconds.

  • Continue will make the scene immediately continue.

  • Wait will make the scene wait until the exit animation is done.

  • Effects - These options affect how the Character will disappear.

  • Easing sets the tween movement of the character. More information can be seen in Easing Effects page.

  • Animation sets the movement of the character.

    • Movement moves the character out of the screen in Left/Top/Right/Bottom direction.

    • Blending allows you to fade out the Character.

    • Masking allows you to mask the Character to a Masks file.

      • Vague the smoothness of the mask edges.

 

Live2D Motion

Sets a new motion for the Live2D character.

  • Character - The Live2D character to set the motion for.

  • Motion - The motion to set.

  • Repeat - Indicates if the motion should be repeated.

  • Duration - The duration of the exit animation in milliseconds.

  • Continue will make the scene immediately continue.

  • Wait will make the scene wait until the exit animation is done.

 

Live2D Motion Group

Sets a new motion-group for the Live2D character. A motion-group is a set of regular motions configured in the Live2D file which can be played randomly or in order.

  • Character - The Live2D character to set the motion-group for.

  • Motion Group - The motion-group to set.

  • Repeat - Indicates if the motion-group should be repeated.

  • Duration - The duration of the exit-animation in milliseconds.

  • Continue will make the scene immediately continue.

  • Wait will make the scene wait until the exit animation is done.

  • Fade-In - Adds a fade-in time for motions. If set to 0, the motion will be displayed instantly.

 

Live2D Expression

Changes the face expression of a Live2D character.

  • Character - The Live2D character to change the face expression for.

  • Expression - The new Live2D face expression to set.

 

Live2D Move

Moves a Live2D character to a specified position on screen.

  • Character - The Live2D character to move

  • Model - The Live2D model to use.

  • Position - The target-position of the Live2D character.

  • Predefined - Allows to select one of the predefined positions configured in Database > System > Predefined Object Positions.

  • Direct - Allow to set the target-position using the graphical editor.

  • Calculated - Allows to calculate the target-position by variable.

  • Duration - The duration of the move-animation in milliseconds.

  • Continue will make the scene immediately continue.

  • Wait will make the scene wait until the move-animation is done.

 

Live2D Parameter

Sets or animates a model parameter of a Live2D character to a specified value.

  • Character - The Live2D character to animate the parameter for.

  • Parameter - The parameter to set/animate. It must be a valid parameter of the Live2D model such as PARAM_ANGLE_X, PARAM_EYE_L_OPEN, etc.

  • Value - The target value to set.

  • Duration - The duration of the animation in milliseconds.

  • Continue will make the scene immediately continue.

  • Wait will make the scene wait until the animation is done.

  • Easing sets easing algorithm for the animation. More information can be seen in Easing Effects page.

 

Live2D Settings

Allows to change the settings of a Live2D character such as default idle animation and eye blinking.

  • Character - The Live2D character to change the settings for.

  • Breath Intensity - The intensity of the breath-animation, ranges from 0.0% to 100.0%. If 0.0% is specified, the breathing will be disabled.

  • Idle Intensity - The intensity of the default idle-animation, ranges from 0.0% to 100.0%. If 0.0% is specified, the idle-animation will be disabled.

  • Lip-Sync Sensitivity - The higher the value the more the mouth-openness will be influenced by the volume of the voice. If 0.0% is specified, the mouth-openness will not be influenced even at high voice volume.

  • Eye Blink  - Allows you to configure eye-blinking

  • Enabled - Enabled/Disable default eye-blinking animation.

  • Interval - The max. time-interval between two blinks in milliseconds.

  • Closed Motion - The time in milliseconds how long the eyes stay closed.

  • Closing Motion - The time in milliseconds how long the close-animation of the eyes will take.

  • Opening Motion - The time in milliseconds how long the opening-animation of the eyes will take.

 

Live2D Defaults

Allows to override default values for Live2D specific commands.

  • Appear Duration is the duration of the join-animation in milliseconds.

  • Disappear Duration is the duration of the exit-animation in milliseconds.

 

  • Appear Effects - Easing is the easing settings used for the join-animation.

  • Appear Effects - Animation is the animation settings for the join-animation.

 

  • Disappear Effects - Easing is the easing settings used for the exit-animation.

  • Disappear Effects - Animation is the animation settings for the exit-animation.

  • Display - Z-Order is the Z-Index of a Live2D character.
