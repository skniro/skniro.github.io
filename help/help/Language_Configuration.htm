Language Configuration

 

The Language Configuration tab allows you to localize your games with ease.

This allows you to import/export the texts to CSV files for collaborations.

 

You can rearrange the entries by clicking the column tabs.

 

Language Profiles

A list of available languages for your visual novel. For additional functions, refer to the Context Menu page.

You can easily adjust the width of the text by dragging the | to view an entire text.

 

  • Database - Allows the user to translate texts found in the Database.

  • Scenes - Allows the user to translate texts found in Scene Contents.

  • Default - The original text to be translated.

  • Translation - By double clicking, it will open the Translation window. This is where you can find the translated texts.
  • Info - Additional Information is displayed here; For example, the set character's name for that line of dialogue.
  • ID - The Index (ID) of that specific text. This is intended for advanced users.

 

Custom Texts

Since texts are automatically taken from the scene and database, you cannot add or delete entries. This is a problem for scripts and other plugin-related materials.

 

Custom Texts is the way to include those texts in the language configuration. To add a text entry to the Language Configuration, do the following steps:

 

1. Highlight the String you want to add to the Language Configuration. Make sure to include the quotes(")! Press Tools -> Developer Tools -> Insert New String

 

2. A window named New Phrase will appear. Here you can enter the Info-Text displayed in the Info-Column in Language Configuration. Can be used to make it more clear for a translator where in the game the text is used.

 

3. With this, you do not have to move away from the editor to add more strings. But if you check back to the Language Configuration tab, the custom text is included.

 

New/Edit Language Profile

The contents of a Language Profile option.

  • Name - The name of the language profile
  • Code - The code of the language you're adding. The user can use any code they want.
  • Flag Icon - Double click on the tiny icon to change the language flag.
  • Writing Direction - The writing-direction of the language. At time of writing, this option has no effect but can be accessed via scrips.
  • Word Wrap - The word-wrap technique used for this language. (The list can be extended. As a developer, just check Constants script in Extension Editor.)

 

Additional Information

Here are some important things you need to know:

  • You cannot  delete or Import/Export CSV the base language; By default it is English.

  • You cannot change which language profile is the default language. Buy you can edit the default language profile.

  • Language code is useful for translating image files. For example:
    You have an image called list.png that is translated to Japanese. You need to save the translated image to filename+_code for its filename (ex. list_jp.png).
    The game will automatically load that image instead of the original if a language with code "jp" was selected.
