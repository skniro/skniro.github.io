Example #1: A Simple Counter

For this tutorial, we'll make a fairly simple example of how to use the Condition command. For this tutorial, we'll be using a Number Variable. First, let's take a look into this flowchart:

 

 

As to how this reads, the flowchart goes with the logic, if Counter is equal to 6, the process will end. Else add +1 to the Counter and check again.

 

So what we will need to do to translate it in VNMaker is as follows:

  • Let's add Message Box Visibility command and set the Visible to Yes. This is so we can use a Show Message command to show the value of the variable we're using.
  • Create a Label called 'start'.
  • Add  Basic -> Condition command. Click the Variable command and change the Scope to Local. Let's name it Counter. Change the Value to 6 afterwards.
  • Afterwards, let's add Basic -> Idle command. This will pause the scene indicating the condition has been achieved. Assuming that the command isn't tabbed for any reason, you can press the Right Arrow () key to put a command under the Condition. If you accidentally tabbed too much, just press the Left Arrow() key.
  • Let's add Basic -> Else command. This will allow us to put another condition if the first condition hasn't been met yet.
  • Next, we'll use a Show Message and use a text code, {LN:1}. This means that whatever is the value in Local Number Variable #1 will be shown in the Show Message command.
  • After that, we'll use a Number Variable and set the Variable Scope to Local Number #1. Set the Operation to Add and Source to Value. Finally set the Value to 1.
  • At the end of the command, add a Jump to Label to start and make sure it's not tabbed.

 

If you did everything correctly, it should look like this.

 

If you use Play Test, you will see the number ticking up to five. We made a simple counter!

 

Example #2: Opening a Treasure Chest

For our next example, we are going to make a small text prompt that asks if we want to open a treasure chest. It will give a different response if you already opened the chest. For this tutorial, we'll be using a Switch. Here's a quick flowchart that explains how this will work:

 

  • As usual, let's add Message Box Visibility command and set the Visible to Yes.
  • Since this is an item that is exclusive for this one scene, we can safely use a Local Switch. This means that switch is only usable on that specific Scene. So let's click the Switch section and change the Scope to Local.
  • Then name the switch to something like 'Hookshot' and press OK.
  • Create a Label called 'start'.
  • Then let's add a Show Message command that says, "There's a treasure chest here. Should we open it?"
  • Now let's add two Add Choices that says Yes and No respectively. Don't forget to add a Show Choices at the end.
  • For the Yes Choice, we'll use the Label ItemGet. For the No Choice, we'll use the Label Ignored.
  • Now, we should add a Label named ItemGet. Put a Condition command under it.
  • Make sure that the Condition's If is set to Switch, Variable Local Switch #1 and is Equal to ON.
  • Let's add a Show Message that says, "We already got the hookshot."
  • After that we will add an Else command.
  • Since we made a Switch earlier, let's drag the Switch under the Else command.
  • Let's add a Show Message that says, "We got a hookshot!"
  • Then we will add a Jump to Label named Start.
