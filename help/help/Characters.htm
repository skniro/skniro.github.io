Characters

 

 

General

  • Name - The name of the character.  

  • Color - When in Novel (NVL) mode, the message text will be displayed with the color on what you assigned here. When in Adventure (ADV) mode, only the character's name will be displayed with that color.

 

Voice Settings

  • Time Talking-Animation to Voice-Volume - If enabled, the talking animation will be synchronized based on the voice clip's volume.
    For Live2D Models, the width of the mouth depends on the voice-volume.

 

Live2D Settings

  • Lip-Sync Sensitivity - The higher the percentage, the more the mouth-openness of the character will be influenced by voice-volume. If 0.0% is specified, the lip-sync is disabled.

  • Breath Intensity - The intensity of the breath-animation. If 0.0% is specified, the breathing is disabled.

  • Idle Intensity - The intensity of the idle-animation. If 0.0% is specified, the default idle-animation will be disabled.

 

Parameters

Parameters are local number, switches or text values that only exists for that designated character. They can be accessed at runtime using the Set Parameter and Get Parameter Scene Commands.

 

  • Name - The name of the parameter.

  • Number - Defines a number-value as parameter. This is useful if you want to add stats to your characters for life simulation games.

  • Switch - Defines a switch-value as parameter.

  • Text - Defines a text-value as parameter.
