Parallax Scrolling

 

Parallax Scrolling is a visual technique in which layers of images are moving in different speeds to create an illusion of depth.

 

 

Preparation Work

First, all the images must be separated into layers and will move at different speeds. This involves the Foreground, Middleground and Background.

The Background would move the slowest.

The Middleground will move in-between the Foreground and Background speed.

And here's another Middleground that doesn't move.

The Foreground will move the fastest.

 

It's important to remember that the images must be created for 'looping'. This means that if you put the image side by side, it will still look seamless.

 

Implementation

 

  • First, we need to set all the background layers. Make sure to select Yes to Loop Horizontal.
    • You have to add a Wait on the last Change Background command. Otherwise, they will scroll immediately before the Loop effect takes place.
  • Then we will need to add the Scroll Background command.
    • Notice how the Duration is set to 0ms. This means the background will scroll infinitely.
    • Speed X and Speed Y represents how fast they will scroll. You can use decimal values. The higher the number, the faster they will scroll.
    • You can use Easing Effects to affect how their scroll animation will move. I used different ones in random and settle with one that I felt just right.

 

And that's about it! You can create the following visual effect with the very same commands!
