Custom Controls


To create database categories or custom views/windows, Visual Novel Maker already provides a powerful set of standard UI controls like Buttons, Check Boxes, Text Fields, Tables, etc. However, if you are developing a huge or very specific extension you maybe want to make an own UI control to give the user an easier and more efficient way to use your extension.

 

In Visual Novel Maker, you can develop custom UI controls using Objective-J files and the Cappuccino + VN Maker Objective-J APIs.

Example: Slider Control

To create a new custom UI control, just create a new Objective-J file. The name of the Objective-J file must match the name of the class definition. You can checkout the EXSliderItemView example in Extension Editor > Scripts > Custom Controls which is shipped out with Visual Novel Maker by default. It is an example how to implement a slider control.
