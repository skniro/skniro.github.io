Adding Characters and Expressions

 

In this section, we'll teach you how to add your characters in Visual Novel Maker!

 

 

Creating Folders

  • Click the Database Tab.
  • Select Collections -> Characters
  • Let's make a few folders! As mentioned in Make a List of Characters, we will create several that separates all the characters by order of importance. Simply press Right Click -> New Folder.
  • Click the New Folder in order to rename it.
  • Depending on the type of story you're doing, you might only need a few folders. For the sake of simplicity, I only made three folders.

If you wish to use an automated importer. Simply read the Expression Importer page.

 

Manually Adding Expressions

If you wish to manually add expressions, the process is similar to Characters.

 

  • Read the Importing Resources page.
  • Much like Characters just create folders that separate them from each other.
  • Select a folder, in this case Protagonist, and then Right Click -> New Record.
  • Click on the New Record and press name to rename it.
  • Your database should look something like this:
  • Simply press Add and select the image file you want to display for that specific emotion. If done correctly, the database should display your character expression like so:

  • Keep doing this to all your characters! And you will have all the character expressions set!

 

EXTRA: Setting Thumbnails for your characters

As a nifty way to show which character is currently on scene, go back to Characters page.

  • Click the desired character.
  • Click Default Expression and select which expression you want the character to display by default.
