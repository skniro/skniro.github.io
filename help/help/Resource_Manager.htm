Resource Manager

 

Visual Novel Maker allows you to import your own resources. Simply press the Resource Manager () icon in The Toolbar.

All resources must be managed in the Resource Manager. Resources can be imported (loaded into the project), exported (written to file), or deleted in this window.

 

 

(1) Main Folders

The folders containing all of the game files.

For more information on what file types are accepted, refer to the Resource Standards.

Characters Files used for Characters..
Backgrounds Files used for Backgrounds..
Pictures Files used for CGs, Animations, System, Picture and UI..
Icons Files used for Icons like for language selection.
Masks Files used for Masks.

Fonts

 Files used for Fonts.

Music

 Files used for Music.

Sounds

 Files used for Sound Effects.

Live2D

 Files used for Live2D.

Movies

 Files used for Movie.

 

(2) Files

The contents of each folder.

 

(3) Import

Add a resource to the specified folder. Importing resources does not require a prefix or suffix.

  • Prefix - adds a specified keyword at the beginning of the filename of an imported resource.
  • Suffix - adds a specified keyword at the end of the filename of an imported resource.

 

(4) Export

Make a copy of the selected file and save it to another location.

You cannot export a folder and they are ignored by default. You must select files to be exported.

 

(5) Delete

Deletes the selected resource..

 

(6) DLC

Imports a resource directly from the DLC folder.

 

(7) Check out latest DLCs

Opens the Steam store page for Visual Novel Maker DLCs.

 

(8) Preview Window

A small preview window to display the contents of the file selected. You can double-click it to get an original size preview of the selected image.

 

Context Menu

You can create new logical groups as well as rename resources via context menu.

 

New Group

Creates a new logical group with the specified name. A group is only a logical element for organization purposes, it is not a physical folder on disk which is why you cannot have two or more resources with the same name even if they are in different groups.

 

Delete

Deletes the selected resource.

 

Properties

Opens the properties of the currently selected resource which allows you to change the name of the resource.

 

  • Name - The file name of the resource. You can change the name of a resource anytime.
  • Rename All References - If checked, VN Maker will walk through all your scenes and database records and tries to automatically find and rename/correct all places where the resource has been used to match with the new file name. However, there are special cases which are not covered such as text variables and scripts. You can check the Debug Console in case of errors or unexpected results.
