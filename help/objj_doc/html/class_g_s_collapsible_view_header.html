implementation

Initializes a new collapsible view header with the specified frame.

Parameters
aFrameThe frame.
collapsibleIndiciates if the view-header should contain an arrow-button to collapse the view's content.
aDescriptorA descriptor which contains more info about the CSS classes to use for the header. The following properties are supported: (All properties are optional)
  • headerCssClass - The CSS class to use for the view-header.
  • headerTextCssClass - The CSS class to use for the view-header text.
  • headerLineCssClass - The CSS class to use for the view-header bottom-line.
- (void) setDelegate: (id) 
implementation

A delegate to get informed about arrow-button clicks.

The following methods are called:

// Called if the arrow-button has been moved to collapsed position. The view should collapse it's content
// now.
- (void)headerCollapse:(GSCollapsibleViewHeader)aViewHeader
// Called if the arrow-button has been moved to expanded position. The view should expand it's content
// now.
- (void)headerExpand:(GSCollapsibleViewHeader)aViewHeader
