GSFile Class Reference

This is not a single class but a collection of file/directory functions you can use read and write files or to create directories. More...

Detailed Description

This is not a single class but a collection of file/directory functions you can use read and write files or to create directories.

You can Node.js API anytime but it is recommended to use the functions described here whenever possible.

// Reads a text file from the specified location on disk and returns
// it's contents as string.
// filePath - The location of the file to read.
function GSFileRead(filePath : string) : string;
// Reads a binary file from the specified location on disk and returns
// it's contents as a binary string.
// filePath - The location of the file to read.
function GSFileReadBinary(filePath : string) : string;
// Reads a file from the specified location on disk and returns
// it's contents as a Node.js Buffer object.
// filePath - The location of the file to read.
function GSFileReadBuffer(filePath : string) : Buffer;
// Gets the last modification time of the specified file.
// filePath - The path of the file to get modification time for.
function GSFileGetModifiedTime(filePath : string) : number;
// Renames a file.
// oldPath - The path to the file to rename
// newPath - The renamed path to the file
function GSFileRename(oldPath : string, newPath : string) : void;
// Indicates if the app has write-access to the specified location on disk.
// path - The path to check.
function GSFileCanWrite(path : string) : boolean;
// Indicates if the specified file is a folder.
// filePath - The path to file to check.
function GSFileIsDirectory(filePath : string) : boolean;
// Writes a Node.js Buffer into a file on disk.
// filePath - The location of the file to write to.
// buffer - The Node.js Buffer to write into the file.
function GSFileWriteBuffer(filePath : string, buffer : Buffer) : void;
// Writes a text into a file.
// filePath - The location of the file to write to.
// content - The text to write into the file.
// encoding - The encoding of the text. Can be omitted.
function GSFileWrite(filePath : string, content : string, encoding : string) : void;
// Appends a text to a text file.
// filePath - The location of the file to append to.
// content - Text to append to the text file.
function GSFileAppend(filePath : string, content : string) : void;

// filePath - The location of the file to write to.
// data - The binary-string to write into the file.
function GSFileWriteBinary(filePath : string, data : string) : void;
// Deletes a file on disk.
// filePath - The location of the file to delete.
function GSFileDelete(filePath : string) : void;
// Copys a file.
// source - The location of the file to copy.
// target - The new target location of the copied file.
function GSFileCopy(source : string, target : string) : void;
// Copys a file asynchonously.
// source - The location of the file to copy.
// target - The new target location of the copied file.
// callback - The callback function to call after the copying-process has been done.
function GSFileCopyAsync(source : string, target : string, callback : Function) : void;

// sources - An array of file paths to copy
// destinationPath - The path to the destination of the copied files.
// callback - The callback function to call after the copying-process has been done.
function GSFileMultiCopy(sources : string[], destinationPath : string, callback : Function) : void;
// Checks if a file exists on disk. Returns TRUE if the file exists. Otherwise FALSE.
// filePath - The path to the file to check

// path - The location of the folder to create.
function GSDirectoryMake(path : string) : void;

// path - The location of the folder on disk. All non existing parent-folders are automatically
// created.
function GSDirectoryMakePath(path : string) : void;
// Gets a list of all file names of the specified folder-path.
// path - The folder-path to get the contents of.
function GSDirectoryContents(path : string) : string[];
// Deletes the folder at the specified location on disk. The folder doesn't need to
// be empty and can contain other files and folders.

function GSDirectoryCopyAsync(source : string, target : string, callback : Function) : void;
