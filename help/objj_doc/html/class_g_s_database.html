Detailed Description

The database stores game data such as characters, character expressions, animations, etc.

for a project. To access the database of the current project, just use the defaultDatabase (GSDatabase) method. It is not recommended to create a new custom instance of this class.

To access database records such as characters, you can use the following example code:

// Get the database instance of the current project.
var database = [GSDatabase defaultDatabase];
// Get all character records. The ".characters" is the data record view category/attribute name.
// For custom database categories, just use your category/attribute name here.
var characters = [database recordCollections].characters
// Print the name of character 0001 on debug console
console.log(characters[0].name);
