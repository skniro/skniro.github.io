Detailed Description

This allows to play music and sounds for preview purposes.

It used for the Music Player and Music/Sound selection dialog box.

Method Documentation

- (GSAudio) createReference
implementation

Creates a new audio object reference fro this audio object which is useful to play a sound multiple times simultaneously.

It is important that this audio object has been already loaded and decoded.

Returns
The audio object reference.
- (void) decode
implementation

Starts the decoding process.

If the audio object is already decoded, the audioDidDecode:(id)aSender method is immediately called on the delegate (if set).

- (void) initWithAudioBuffer: (GSAudio)  anAudioBuffer

- GSAudio: (CPObject) 
implementation

Sets the audio object's delegate.

The following delegate-methods are called:

  • audioDidLoad:(id)aSender - Called if the audio object has been loaded successfully.
  • audioDidError:(id)aSender - Called if an error occurred during the loading process.
  • audioDidDecode:(id)aSender - Called if the audio decoding has been finished.
  • audioDidDecodeError:(id)aSender - Called if the audio decoding has been aborted with an error
Parameters
aDelegateThe delegate to set.
