Detailed Description

The item-view factory is used create item-views depending on type.

You also have to register your custom item-views here using registerItemView:forTypeName: (GSOItemViewFactory) method to make them available in your custom views. An example Objective-J File in Extension Editor would look this:

@implementation MyItemView : GSOItemView
{
// Properties....
}
// Implementation....
@end
[GSOItemViewFactory registerItemView:MyItemView forTypeName:@"MyItemView"];

After that, you can use your custom item-view in your views as follows:

{
"type": "MyItemView",
"frame": [0, 0],
......
}
