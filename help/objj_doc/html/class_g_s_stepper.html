Detailed Description

A stepper-control combines a CPTextField and a CPStepper in one number-input control with up/down buttons on the right end.

The following example creates a new stepper-control:

// Create a new stepper-control with 200px in width.
var stepper = [[GSStepper alloc] initWithFrame:CGRectMake(0, 0, 200, 0)];
// Allow floating-point values
[stepper setNumberFormatterStyle:CPNumberFormatterDecimalStyle];
// Set minimum, maximum and current number value
[stepper setMinimum:1];
[stepper setMaximum:1000];
[stepper setFloatValue:10.5];
// Add the stepper-control to the content/parent view.
[contentView addSubview:stepper];
