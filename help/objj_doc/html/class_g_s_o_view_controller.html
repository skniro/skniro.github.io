Detailed Description

A view controller allows you to easily create and manage a GSOView from a raw view-descriptor, document or UID of a document.

It also manages the correct creation of the view including caching-mechanisms. The following example creates a GSO view from a Custom View:

// Create view controller from Custom View
var controller = [[GSOViewController alloc] initWithUid:@"<custom-view uid>"];
// Access GSOView object
var view = [controller view];
// Add to parent-view or any other kind of view.
[parentView addSubview:view];
