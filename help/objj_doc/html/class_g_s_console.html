Detailed Description

The console allows you to log debug message on VN Maker's debug console.

You access the debug console via View > Debug Console. It is recommended to use the default debug console using debugConsole (GSConsole) method. You can also just use console.log, console.warn or console.error since that functions will delegate the log message to the default debug console.
