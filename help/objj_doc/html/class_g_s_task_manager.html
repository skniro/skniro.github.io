Detailed Description

All tasks are managed by the task manager.

Each task needs to be added to the task manager first before it can be executed. The task manager also takes care of progress display and error display. You can use defaultManager (GSTaskManager) method to access the default task manager instance.

If you add new task to the task manager, the task manager will set itself as the task's delegate to catch and display status, message and error information correctly. An example how to create and execute a custom task looks like this:

var task = [[CustomTask alloc] init];
[[GSTaskManager defaultManager] addTask:task];
[[GSTaskManager defaultManager] runTasks];
