Understanding Choices and Labels

 

In this page, we will teach you how to use choices and labels in Visual Novel Maker. The Label is a command that allows you to bookmark a specific part of the scene and 'Jump' to it whenever you want to. Here is a quick demonstration on how to set up.

As you can see, we added a Label called, 'start' at the very beginning. After the Show Message, we added a Jump to Label command that goes back to 'start'. Because of this, the scene loops infinitely.
