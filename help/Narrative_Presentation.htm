Narrative Presentation

 

It is important for us to decide on how we should deliver the narrative to the player/reader. It is not to say that you have to be tied down to just one type of presentation. But it would help to understand why certain stories are illustrated this way. There are many types but we'll only be discussing the just a few of them.

 

Novel Mode (NVL)

Novel Mode is a form of narrative display that is similar to reading a book --hence its name. Normally, this type of presentation focuses on a character's stream of thoughts and observations. While most Novel type games have text across the screen, it does not necessarily mean that it has to be.

 

This type of narrative is perfect for writers. It's more focused on traditional writing (albeit on using present tense instead of past tense). NVL mode allows you to focus more on sounds and your writing with minimal art.

 

Adventure Mode (ADV)

Adventure mode is the type of narrative display that focuses on character dialogue and minimal description for its pacing. Most games tend to use this type of narrative.

 

This type of narrative has equal amounts of focus on art and writing. It is discouraged to use third person narration upon using this mode.

 

Animated Storyboard (Animatic) 

An animatic is a rough storyboard that focuses on visualizing an animated movie before it begins production. However it has been adapted into visual novel format by some developers. There is a narrative but it is laid out like a film.

 

 

Motion Comic  

Motion Comic is a type of narrative that emulates a comic's type of storytelling. This type of narrative is heavily visual as it focuses more on animations, transitions and almost strictly character dialogue.

 

This is by far the rarest to be seen due to the production values involved into creating it.
