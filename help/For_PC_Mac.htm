For PC / Mac

 

Testing & Deployment for PC / Mac is very easy. This topic will guide you through the necessary steps.

 

Testing on PC / Mac

 

If you want to do a quick play-test, you can just press the play-button from The Toolbar or choose Game > Play/Play in Fullscreen from The Menubar. That's it!

 

Deploy for PC / Mac

 

By default, VN Maker supports deployment for Windows, macOS, Linux Ubuntu and SteamOS operating systems. You can deploy your game for any of that platforms even if your operating system doesn't match with the deployment platform. So if you are using Windows, you can still deploy for macOS or Linux and vice-versa.

 

To deploy your game, you can either choose Build For > [target-platform] from The Menubar for a quick test-build or use the Create Game Package wizard to create a ready-to-ship game package.

If you want to take more control over how your game will be packaged, take a look at Build & Run documentation.
