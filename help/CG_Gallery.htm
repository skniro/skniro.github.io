CG Gallery

 

 

General

  • Name - The name of the CG Gallery entry.

  • Listed - If checked, there will be a [?] thumbnail in the In-Game CG Gallery. Otherwise, it won't appear in the CG Gallery 'locked CGs' list.

  • Chapter - This is the setting for which chapter will the CG be listed.

 

Graphic & Thumb

  • Graphic - The full size image of the CG.

  • Thumb - A preview of the CG and will be displayed in the CG Gallery selection. It should be 1/8 of the original CG size. If not, it will be automatically scaled to that size but still consumes the same amount of memory as the original CG. So its recommended to provide an already scaled down image here.
