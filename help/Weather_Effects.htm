Atmosphere and Effects

 

For this tutorial, we're going to explore some commands that will help us establish atmosphere for our Visual Novel. Visual Novel Maker utilize videos and sometimes with alpha transparency to display effects such as weather. It primarily uses the Show Movie command. You can find the following effects under your game's Movie folder.

Example #1: Making a Rainstorm

 

Here's how it was implemented:

 

  • Add a Show Movie commands. For more details on what each separate parts does, please see the link.
  • Since this is a Rainstorm, we will use the Strong Rain (rain2) movie clip. I added a fog just for dramatic effect. It is not necessary.
  • If the video files are the same size as your screen resolution, just leaving the position to Predefined -> Top Left should be fine.
  • The movie files have the addition (add) blend mode so they will blend in with the current scene. This is used especially when a movie file has a black background.
  • Play Music command is for the ambient music. I lowered the Volume to 51% just so it wouldn't be too overbearing.
  • Play Sound is for the thunder sound effect. Unfortunately, I cannot find a sound effect that is free to use that is just the thunder alone. So it needed a bit of delay.
  • Wait and Flash Screen is to time the thunderclap.

 

If you want a thunderclap to happen once every few seconds, you can use a Common Event.

 

Example #2: Sparkling Senpai

Here's the scenario, Emma just saw Ace for the first time and it was love at first sight! What would be the natural reaction? In true Shoujo fashion, sparkles of course! Flowers sold separately.

 

Here's how it was implemented:

 

  • Add a Show Movie commands. For more details on what each separate parts does, please see the link. The same as the first example's situation, Blend Mode to addition (add) since the movie file has a black background.
  • I used a Zoom Movie command since the video is smaller than my screen resolution. The position can be left at Predefined Top Left position thanks to this.
  • I added Shake Background and Background Effect to show element of surprise and dizziness on Emma's part.
